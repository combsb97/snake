# Snake in C with ncurses

This project is a terminal graphics implementation of the classic game, Snake. Written in C this project uses the ncurses library.

## Getting Started

### Dependencies

* ncurses

### Executing program

* compile
```
$ gcc snake.c -o snake -lncurses
```
* run (Linux)
```
$ ./snake
```
